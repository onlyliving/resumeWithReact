# RESUME

-   웹 프론트엔드 개발자, 김소희 이력서입니다.
-   Tech Stack : React.js (함수형 컴포넌트), TypeScript

## 실행하기

```bash
# after source code download

# npm download
$ yarn install

# run dev server
$ yarn dev
```

## 히스토리

-   구 이력서 페이지는 React 클래스형 컴포넌트와 JavaScript로 구현을 하였습니다.
-   현 이력서 페이지는 React 함수형 컴포넌트와 TypeScript로 재구성 하였습니다.
